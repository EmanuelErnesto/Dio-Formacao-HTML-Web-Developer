Início

O HTML foi criado por Tim Berners-Lee com o objetivo de ser uma linguagem de marcação utilizada para estruturação de páginas Web. Através do HTML, foi possível criar páginas que poderiam ser conferidas a partir do protocolo HTTP. Muitos comumente o associam com uma linguagem de programação, mas fato é que ele foi concebido e pode ser definido apenas como uma linguagem de marcação de texto.

A primeira página criada também por Tim Berners-Lee foi uma que descrevia como funciona a World Wide Web (mais conhecida pelas siglas WWW). Você pode conferi-la aqui

Tags Comumente usadas

O HTML possui um conjunto de tags que trazem uma grande diversidade de opções para o programador estruturar o site. Desde Hyperlinks com a tag "a" até formulários de texto e números com a "input" é possível explorar uma vasta gama de possibilidades. Para aquele que irá estruturar o FrontEnd, ou o "lado do cliente" é possível também trazer, mesmo sem o uso de um CSS, uma pequena estilização ao site. Com tags como "strong", "u" e "i" é possível deixar em negrito, com o "underline" e em itálico, respectivamente. Isto torna o HTMl uma poderosa ferramenta para criação da estrutura de sites, e em conjunto com o CSS que trás uma maior gama de estilizações, as possibilidades são praticamente intermináveis. Abaixo estarão listados alguns exemplos de tags e suas funcionalidades.

Tag de parágrafo

A tag "p" serve para delimitar onde começa e termina o páragrafo de um texto. É possível ter várias tags p em uma página, assim como o HTML não possui algum tipo de erro ao utilizar mais de uma mesma tag ao mesmo tempo. Basta apenas deixa-las devidamente separadas por sua sintaxe base, que vem a partir da tag de abertura e a tag de fechamento

Tags para listas

As listas no HTML nos permitem trazer de modo ordenado ou desordenado um grupo de itens. A lista a ser usada depende da necessidade do programador na hora do desenvolvimento de sua webpage, sendo bem flexível neste sentido.

As listas ordenadas (ol) possuem uma ordenação clara para os seus elementos da lista (ou list items) tornando mais fácil uma visibilidade da ordem dos elementos.

  1. Esse é um item da lista ordenada
  2. Este é outro item da lista ordenada
  3. Esse é mais um item da lista ordenada.

Assim como na matemática, "a ordem dos fatores não altera o produto" então se quisesse mudar de linha qualquer uma dessas 3 frases, seria possível, dependendo apenas dos requisitos e necessidades específicos para o site.

Quanto às listas desordenadas, elas são, como sugere o nome, elementos que não possuem uma ordem clara, havendo por padrão apenas um ponto escuro do lado esquerdo de seus elementos. Este ponto é uma estilização padrão do HTML, podendo, com auxílio do CSS, ser modificado e estilizado para um maior embelezamento do site. Tanto as ol quanto as ul possuem suas tags de abertura e fechamento, assim como cada um dos itens das listas, que são delimitados pela tag li (list items)

  • Esse é um item da lista desordenada
  • Este é outro item da lista desordenada
  • Esse é mais um item da lista desordenada

Tag para Hyperlinks

Para hyperlinks a tag mais comumente utilizada é a 'a', cujo possui uma tag de abertura e fechamento, bem como possui uma referência e um alvo (ou target). A referência é o caminho para onde o Hyperlink irá encaminhar, enquanto o target diz para o navegador se será aberto na mesma aba que a página web que nós construímos, ou se será aberto em outra página. Quando o objetivo é manter o foco do usuário na página atual, então o ideal é utilizar o parâmetro _self dentro do target para dizer para o navegador que queremos que o link leve para o caminho desejado na aba atual. Caso o objetivo seja que o usuário seja reencaminhado para outra página, é passado como parâmetro para o target o _blank. Antigamente haviam outros dois parâmeteos para o target, mas atualmente eles estão em desuso, sendo possível vê-los apenas em sites mais antigos onde seja necessária alguma manutenção, por exemplo.

É possível definirmos esses links para tanto páginas externas como internas, sendo por isso que a tag 'a' costuma ser chamada de 'tag ancora'
